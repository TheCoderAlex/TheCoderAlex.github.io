增加看板娘插件

使用 npm 安装：
在 Hexo 项目的根目录运行命令：
npm install --save hexo-helper-live2d
3.0.1 版本存在的 bug：
存在代码注入 bug，与 hexo-tag-dplayer 插件产生冲突，导致相关脚本无法加载。
解决方法：在项目中使用 npm 命令手动卸载 “hexo-tag-dplayer”: “^0.3.1”。

增加爱心点击效果

在/themes/icarus/source/js/src下新建文件clicklove.js，接着把该链接下的代码拷贝粘贴到clicklove.js文件中。

在/theme/icarus/layout/layout.ejs的之后增加语句:
<script src="/js/clicklove.js"></script>
大概就是这个样子了，感谢大佬们在前面铺的路，为我们减少了不少功夫！
